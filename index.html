Linguaggi di Programmazione

Kilian Contartese

HTML

HTML un linguaggio per la costruzione di siti internet. È costituito da un insieme di comandi che permettono di definire la struttura e l'aspetto di un sito o di inserire oggetti (immagini, tabelle, moduli).

Usalo
CSS

Il CSS è un linguaggio usato per definire la formattazione di documenti HTML, XHTML e XML, ad esempio i siti web e relative pagine web.

Usalo
JAVASCRIPT

JavaScript è un linguaggio di scripting lato client utilizzato per rendere interattive le pagine web. JavaScript, insieme ad HTML e CSS, è una delle tecnologie principali della programmazione web front end.

Usalo
C++

In informatica C++ è un linguaggio di programmazione general purpose sviluppato in origine da Bjarne Stroustrup nei Bell Labs nel 1983 come evoluzione del linguaggio C inserendo la programmazione orientata agli oggetti.

Usalo
JAVA

Java è un linguaggio di programmazione e una piattaforma di elaborazione sviluppati da Sun Microsystems nel 1995. Esiste un numero notevole di applicazioni e siti Web, in aumento ogni giorno, che funzionano esclusivamente se è stato installato Java. Java è veloce, sicuro e affidabile.
